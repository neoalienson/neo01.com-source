# Chronometer

A stopwatch tool to measure elapsed time with lap support.

## Features

- **Start/Stop/Pause** - Full stopwatch controls
- **Lap Recording** - Record lap times while running
- **Reset** - Reset the stopwatch to zero
- **Millisecond Precision** - Accurate to the millisecond
- **Multiple Laps** - Track and display all lap times

## Usage

Click Start to begin timing. Use Lap to record lap times while the stopwatch is running. Pause to temporarily stop, then resume. Stop resets the timer, Reset clears all data.
