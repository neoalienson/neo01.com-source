关于 IPv4 子网计算器

IPv4 子网计算器从 IP 地址和 CIDR 表示法计算网络信息。子网划分将大型网络分割为较小、可管理的网段，以实现高效的 IP 地址分配和网络组织。此工具计算网络地址、广播地址、可用主机范围、子网掩码和可用主机数量。

计算器接受 CIDR 表示法（例如 192.168.1.0/24），其中斜线后的数字表示网络前缀长度。它支持从 /1 到 /32 的所有子网大小，并提供快速访问常见的 Class A（/8）、Class B（/16）和 Class C（/24）网络。结果包括二进制表示以供教育目的。

常见使用场景

网络管理员在设计网络架构、将 IP 地址范围分配给部门或 VLAN、排除路由问题以及规划网络扩充时使用子网计算器。学习网络概念的学生使用它来理解子网划分原理。IT 专业人员验证子网配置并计算新部署的可用地址。

输入带有 CIDR 表示法的 IP 地址，或使用类别按钮进行常见配置。计算器会立即显示所有子网信息，包括网络范围、广播地址和可用主机。所有计算都在您的浏览器本地进行。
