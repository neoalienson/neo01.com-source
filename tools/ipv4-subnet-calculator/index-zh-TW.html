關於 IPv4 子網路計算器

IPv4 子網路計算器從 IP 位址和 CIDR 表示法計算網路資訊。子網路劃分將大型網路分割為較小、可管理的網段，以實現高效的 IP 位址分配和網路組織。此工具計算網路位址、廣播位址、可用主機範圍、子網路遮罩和可用主機數量。

計算器接受 CIDR 表示法（例如 192.168.1.0/24），其中斜線後的數字表示網路前綴長度。它支援從 /1 到 /32 的所有子網路大小，並提供快速存取常見的 Class A（/8）、Class B（/16）和 Class C（/24）網路。結果包括二進位表示以供教育目的。

常見使用情境

網路管理員在設計網路架構、將 IP 位址範圍分配給部門或 VLAN、排除路由問題以及規劃網路擴充時使用子網路計算器。學習網路概念的學生使用它來理解子網路劃分原理。IT 專業人員驗證子網路配置並計算新部署的可用位址。

輸入帶有 CIDR 表示法的 IP 位址，或使用類別按鈕進行常見配置。計算器會立即顯示所有子網路資訊，包括網路範圍、廣播位址和可用主機。所有計算都在您的瀏覽器本地進行。
