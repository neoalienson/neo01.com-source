关于 Defang 工具

Defang 工具通过替换关键字符来安全地中和化潜在恶意的 URL、IP 地址和电子邮件地址，防止意外点击或自动处理。这在网络安全中至关重要，尤其是在报告、电子邮件或文档中分享入侵指标（IOC）时。Defang 使 URL 和 IP 不可点击，同时保持其可读性以便分析。

此工具提供多种 defang 选项：将点号替换为 [.]、将 http 转换为 hxxp、用 [://] 包裹协议以及用括号包裹端口。您可以独立启用或禁用每个选项。Refang 功能反转此过程，在控制环境中需要测试或调查时恢复原始 URL。

常见使用场景

安全分析师在记录恶意软件活动、分享威胁情报报告、在公共论坛发布 IOC、创建事件响应文档以及与团队沟通可疑域名时使用 defang。它防止意外导航到恶意站点，并阻止安全工具自动标记或阻止共享内容。

将您的 URL、IP 或电子邮件粘贴到输入字段中，选择您的 defang 选项，然后点击 Defang。需要时使用 Refang 恢复原始格式。所有处理都在您的浏览器本地进行。
