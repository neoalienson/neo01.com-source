關於 DNS over HTTPS (DoH)

此工具使用 DNS over HTTPS (DoH) 執行 DNS 查詢，它加密 DNS 查詢以保護您的隱私。傳統的 DNS 查詢以明文傳送，允許 ISP 和網路營運商看到您正在查詢哪些網域。DoH 將這些查詢包裝在 HTTPS 中，使它們與常規網路流量無法區分。

支援的記錄類型：

  • A - IPv4 位址記錄
  • AAAA - IPv6 位址記錄
  • CNAME - 規範名稱（別名）記錄
  • MX - 郵件交換伺服器記錄
  • TXT - 文字記錄（SPF、DKIM 等）
  • NS - 名稱伺服器記錄
  • SOA - 授權起始記錄
  • PTR - 指標記錄（反向 DNS）
  • CAA - 憑證頒發機構授權

DoH 提供商：

  • Cloudflare (1.1.1.1) - https://cloudflare-dns.com/dns-query - 注重隱私，最少日誌記錄
  • Google (8.8.8.8) - https://dns.google/resolve - 高效能，全球網路
  • Quad9 (9.9.9.9) - https://dns.quad9.net:5053/dns-query - 注重安全，威脅阻擋
  • 自訂 - 使用您自己的 DoH 伺服器 URL

注意：自訂 DoH URL 必須支援 JSON API 格式（application/dns-json）並允許來自此網域的 CORS 請求。
