HTTPS 経由で DNS をクエリしています...

DNS over HTTPS (DoH) について

このツールは DNS over HTTPS (DoH) を使用して DNS クエリを実行し、DNS クエリを暗号化してプライバシーを保護します。従来の DNS クエリは平文で送信され、ISP やネットワークオペレーターがクエリしているドメインを見ることができます。DoH はこれらのクエリを HTTPS でラップし、通常のウェブトラフィックと区別がつかないようにします。

サポートされているレコードタイプ：

  • A - IPv4 アドレスレコード
  • AAAA - IPv6 アドレスレコード
  • CNAME - 正規名（エイリアス）レコード
  • MX - メール交換サーバーレコード
  • TXT - テキストレコード（SPF、DKIM など）
  • NS - ネームサーバーレコード
  • SOA - 権限の開始レコード
  • PTR - ポインターレコード（逆引き DNS）
  • CAA - 認証局の承認

DoH プロバイダー：

  • Cloudflare (1.1.1.1) - https://cloudflare-dns.com/dns-query - プライバシー重視、最小限のログ記録
  • Google (8.8.8.8) - https://dns.google/resolve - 高性能、グローバルネットワーク
  • Quad9 (9.9.9.9) - https://dns.quad9.net:5053/dns-query - セキュリティ重視、脅威ブロック
  • カスタム - 独自の DoH サーバー URL を使用

注意：カスタム DoH URL は JSON API 形式（application/dns-json）をサポートし、このドメインからの CORS リクエストを許可する必要があります。
