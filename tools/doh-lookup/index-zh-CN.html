关于 DNS over HTTPS (DoH)

此工具使用 DNS over HTTPS (DoH) 执行 DNS 查询，它加密 DNS 查询以保护您的隐私。传统的 DNS 查询以明文发送，允许 ISP 和网络运营商看到您正在查询哪些域名。DoH 将这些查询包装在 HTTPS 中，使它们与常规网络流量无法区分。

支持的记录类型：

  • A - IPv4 地址记录
  • AAAA - IPv6 地址记录
  • CNAME - 规范名称（别名）记录
  • MX - 邮件交换服务器记录
  • TXT - 文本记录（SPF、DKIM 等）
  • NS - 名称服务器记录
  • SOA - 授权起始记录
  • PTR - 指针记录（反向 DNS）
  • CAA - 证书颁发机构授权

DoH 提供商：

  • Cloudflare (1.1.1.1) - https://cloudflare-dns.com/dns-query - 注重隐私，最少日志记录
  • Google (8.8.8.8) - https://dns.google/resolve - 高性能，全球网络
  • Quad9 (9.9.9.9) - https://dns.quad9.net:5053/dns-query - 注重安全，威胁阻止
  • 自定义 - 使用您自己的 DoH 服务器 URL

注意：自定义 DoH URL 必须支持 JSON API 格式（application/dns-json）并允许来自此域的 CORS 请求。
