使用 Chrome 內建 AI API 的實驗性工具。這些工具需要啟用 AI 功能的 Chrome Canary。

<div class="category-header">
  <h2>🤖 文字處理</h2>
</div>

<div class="tools-grid">
  <a href="/zh-TW/ai/summary/" class="tool-card">
    <h3>📝 文字摘要器</h3>
    <p class="tool-description">使用 Chrome 內建的摘要 API 產生文字摘要。可選擇不同的摘要類型、長度和格式。</p>
    <ul class="tool-features">
      <li>多種摘要類型（重點、TL;DR、預告、標題）</li>
      <li>可調整長度（短、中、長）</li>
      <li>輸出格式（Markdown、純文字）</li>
      <li>即時代幣使用追蹤</li>
    </ul>
  </a>

  <a href="/zh-TW/ai/prompt/" class="tool-card">
    <h3>💬 提示 API 遊樂場</h3>
    <p class="tool-description">透過對話介面與 Chrome 內建語言模型（Gemini Nano）互動。實驗不同參數並查看即時回應。</p>
    <ul class="tool-features">
      <li>支援 Markdown 的串流回應</li>
      <li>可調整溫度和 top-k 參數</li>
      <li>工作階段管理和代幣追蹤</li>
      <li>對話歷史顯示</li>
    </ul>
  </a>
</div>

---

*注意：這些工具需要啟用實驗性 AI 功能的 Chrome Canary/Beta。請造訪 chrome://flags/#optimization-guide-on-device-model 以啟用 AI 功能。*
